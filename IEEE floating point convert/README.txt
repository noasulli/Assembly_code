Lab 4: Sorting Floats

-----------
DESCRIPTION

In this lab, the user enters three program arguments in IEEE 754 single precision floating point format.
These numbers will be sorted numerically in ascending order, then printed to the screen in floating point and decimal
format.

-----------
FILES

-
Diagram.pdf

This file includes a block diagram of the program design. The application lucidchart was used to draft the diagram and generate the pdf.

-
Lab4.asm

This file includes the assembly code of the lab.

-----------
INSTRUCTIONS

This program is intended to be run using the MIPS Assembler and Runtime Simulator (MARS). Enter the test case as a program argument and run using MARS.
